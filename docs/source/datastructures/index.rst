===========================
Data Structures
===========================

At the core of ``dynts`` stands :class:`dynts.TimeSeries`, 
the interface class for Time Series. Every timeseries object
is an instance of this interface. The :class:`dynts.TimeSeries`
interface is a subclass of the :class:`dynts.DynData` the base class
of all ``dynts`` data structures.

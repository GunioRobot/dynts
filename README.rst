Quantitative financial timeseries analysis


Requirements
=====================
 * numpy__
 * ply__
 * rpy2__
 * ccy__

Running Tests
=================
Form the package directory::
	
	python runtest.py
	
or, once installed::

	from dynts import runtests
	runtests()
